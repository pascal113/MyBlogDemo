## Envrionment

```
node @7.9.0
db @3.4.0
...

mongodb
python3

sudo apt-get install libsass-dev

sudo apt update
sudo apt install python

sudo systemctl start mongod

注意MongoDB初始化后需要初始化一个admin/admin账户，用于登录后台管理
```

## Run

    git clone git@github.com:Nealyang/React-Express-Blog-Demo.git
    
    npm i
    
    npm start

## Build

    npm run build

## Production
    
    npm run start-prod
